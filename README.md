# 📄 Quote Request PDF Generator

A lightweight, standalone web-based **Quote / Invoice Generator** that allows users to create, preview, and download professional PDF quotes directly from the browser — **no backend, no database, no installation required**.

---

## 🚀 Features

- 🧾 Create professional **Quote Requests / Invoices**
- ✏️ Editable fields:
  - Description  
  - Quantity (QTY)  
  - Unit  
  - Rate  
  - Auto-calculated Amount  
- ➕ Add / remove line items dynamically  
- 💰 Automatic calculations:
  - Subtotal  
  - Discount (%)  
  - Tax / GST (%)  
  - Down Payment (%)  
- 👁️ Live preview before exporting  
- 📥 Download as **A4 PDF**  
- 🖼️ Logo support (loaded locally)  
- 🌐 Website URL included in PDF  
- 📱 Fully responsive (works on phone, tablet, laptop)  
- ⚡ Works **offline** (no internet required)
